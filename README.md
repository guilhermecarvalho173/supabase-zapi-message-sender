# Supabase + Z-API WhatsApp Sender

Script em Python que busca contatos cadastrados no Supabase e envia, via Z-API, a mensagem:

```texto
Olá, <nome_contato>, tudo bem com você?
```

## Setup da tabela no Supabase

Crie uma tabela chamada `contatos`:

```sql
create table contatos (
  id bigint generated by default as identity primary key,
  nome text not null,
  telefone text not null,
  created_at timestamptz default now(),
  sent_at timestamptz
);
```

Exemplo de cadastro:

```sql
insert into contatos (nome, telefone)
values
  ('Maria', '5511999999999'),
  ('João', '5511888888888');
```

O telefone deve estar no formato aceito pela Z-API, normalmente com DDI + DDD + número, sem espaços ou símbolos.

O script busca apenas contatos com `sent_at is null`. Depois que a Z-API confirma o envio, o campo `sent_at` é preenchido automaticamente.

## Variáveis de ambiente

Copie o arquivo `.env.example` para `.env` e preencha:

```env
SUPABASE_URL=https://SEU_PROJETO.supabase.co
SUPABASE_KEY=sua_chave_anon_ou_service_role
SUPABASE_CONTACTS_TABLE=contatos
SUPABASE_CONTACT_NAME_COLUMN=nome
SUPABASE_CONTACT_PHONE_COLUMN=telefone

ZAPI_INSTANCE_ID=sua_instancia
ZAPI_TOKEN=seu_token
ZAPI_CLIENT_TOKEN=seu_client_token_opcional
```

## Como rodar

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute:

```bash
python main.py
```
